### Project Description

- 숫자가 1 또는 2가 나오면 현재 점수 초기화 후 턴을 넘김
- 숫자가 3 ~ 6 은 현재 점수에 나온 주사위 숫자를 더함

- HOLD로 현재 점수 + 누적 점수 => 턴 넘기기
- 계속 주사위 던지기

- 누적 점수 50점을 만들면 이김


### Project EnhanceMent
